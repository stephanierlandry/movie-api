Update User Info This endpoint allows a user to update their info "/update-users/:username/:password/:email" PUT 
      {
      	"username" : "Jon",
      	"password" : "pass1",
      	"email" : "hello@gmail.com",
      	"favorites" : "Girl, Interrupted"
      }
Example: John has updated their info
Response Code: 201
Delete a User This endpoint allows a user to delete their info "/delete-users/:username" DELETE
Example: User steph was deleted.
Response Code: 202
